Secured ATM And Energy Management

The Idea of Designing and Implementation of Security Based ATM theft project is born with the observation in our real life incidents happening around us. This project deals with prevention of ATM theft from robbery so overcome the drawback found in existing technology in our society. We are using MEMS sensor that is accelerometer which is used to detect there is any movement with the ATM machine. This project uses 89S52 controller based system to process real time data collected using the MEMS sensor. If movement detected it will send the message to the police and owner or manager of the bank through the GSM. In this system LCD Display is used to display the message. PIR sensor is used for energy management purpose when any person is entered into the ATM at that time movement is detected by the PIR sensor then only light is ON and energy get saved.
